# LSUS Interactive Campus Navigator (In Progress)

An interactive web-based campus map for Louisiana State University Shreveport.  
This project is designed to help students locate buildings, services, and navigate across campus.

## Current Features
- Interactive campus layout
- Building location markers
- Basic UI for navigation

## Planned Features
- Building information pop-ups
- Route navigation between buildings
- Public transportation integration
- Mobile-friendly interface

## Tech Stack
- React
- TypeScript
- Vite
- Tailwind CSS

## Status
This project is currently under active development.
